```linux
ls                  查看当前所有路径信息
cd + __             跳转到指定路径
:q                  返回终端界面
vim + ___           用vim编辑器打开指定文件
vi + __             用vi编辑器打开一个文档
ifconfig            查看目前设备链接的网路信息-Linux
ipconfig            查看目前设备链接的网路信息-windows
ping + __域名__     访问当前连接速度    linux+windows   使用ctrl+c终止访问
hostnamectl set-hostname __需要修改的主机名__   直接修改当前主机名  不需要重启
hostnamectl         查看当前主机配置信息
hostname            查看当前主机名
vim /etc/hostname   查看并修改主机名        修改的结果会在重启后修改
vim /etc/hosts      查看存放主机地址和名字的文档
```
