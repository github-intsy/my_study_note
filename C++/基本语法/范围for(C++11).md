范围for是一个语法糖
特点->写起来比较简洁
##### 实例:
```c++
    auto array[] = {1,2,3,4,5,6,7,8};
    for(auto e : array) //遍历array数组中的所有元素,将每个元素依次赋值给e,走到array的终点会自动结束   
    {
        e *= 2;
    }
```
    如果想要改变array里面的值,可以使用&
```c++
    for(auto& e : array)
    {
        e *= 2;     //使用别名
    }
``` 
数组在传参以后就会退化成指针,所以
利用数组传参的函数内部不能使用
```c++
    void TestFor(int array[])
    {
        for(auto e : array)         //报错
        {
            cout << array << endl;
        }
    }
```
也就是说语法糖只能对数组进行使用

#### 范围gor的实现
    范围for是由迭代器支持的,也就是说这段代码最终会被编译器替换成迭代器
