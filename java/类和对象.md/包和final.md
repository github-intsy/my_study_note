#### 使用其他类的规则
- 使用同一个包中的类时，不需要导包。
- 使用java.lang包中的类时，不需要导包
- 其他情况都需要导包
- 如果同时使用两个包中的同名类，需要用全类名

#### 构造代码块
1. 写在成员位置的代码块
2. 作用:可以把多个构造方法中重复的代码抽取出来
3. 执行时机:我们在创建本类对象时候会先执行构造代码块, 再执行构造方法
```java
public class Student{
    private String name;

    {
        sout("开始创建对象了");
    }

    public Student(){
        调用方法();
    }
}
```
